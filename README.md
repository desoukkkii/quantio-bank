# SecureBank - Modern Banking System

A complete, beginner-friendly banking web application built with HTML, CSS, and JavaScript.

## Features

- Create bank accounts with initial deposits
- Deposit and withdraw money
- Transfer funds between accounts
- View transaction history
- Real-time balance updates
- Data persistence with localStorage
- Responsive design for all devices
- Beautiful modern UI with animations

## How to Use

1. Download all three files (index.html, style.css, script.js)
2. Put them in the same folder
3. Double-click index.html to open in your browser
4. Create a new account to get started

## Project Structure

banking-system/
├── index.html # Main interface
├── style.css # Styling and animations
└── script.js # Banking logic and functionality


## Banking Operations

- **Deposit:** Add money to your account
- **Withdraw:** Remove money (with overdraft protection)
- **Transfer:** Send money to other accounts
- **History:** View all past transactions
- **Persistent:** Data saves automatically in your browser

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript ES6+ (Classes, Maps, LocalStorage)
- Font Awesome Icons
